# Fatec-tg

**A standard software project.**

## Installation

This project requires the following dependencies:

*   Python 3.7+
*   Git
*   pip

**Instructions:**

1.  Clone the repository: `git clone https://github.com/fatec-tg/fatec-tg.git`
2.  Navigate to the repository directory: `cd fatec-tg`
3.  Install dependencies using `pip install -r requirements.txt`.  (You may need to install dependencies individually if you encounter errors).
